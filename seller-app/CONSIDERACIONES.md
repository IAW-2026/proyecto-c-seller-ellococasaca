# El Loco Casaca - Seller App - Consideraciones 👕

A continuación se detallarán ciertas actualizaciones a tener en cuenta, con respecto a la entrega de la Etapa 2 de la aplicación.

## 🛠️ Actualizaciones
- Se implementó la subida y edición de imágenes de los productos con **Cloudinary**. Si bien en la base de datos se muestra que es posible subir varias imágenes, para los objetivos de la entrega se consideró la subida y edición de UNA IMÁGEN del producto (así como también la eliminación).
- Se modificó visualmente la aplicación, con el objetivo de ser consistente con la aplicación **Buyer App**.
- Se implementó de forma exitosa y ordenada los loadings de cada una de las páginas, con la finalidad de mejorar la experiencia visual del usuario.
- Se modificó la base de datos para poder manejar órdenes de forma correcta, se creó la estructura **Orders**, la cuál representa a la órden de venta, y se se modificó **Sales**, ahora llamada **OrderDetail**, correspondiente al detalle de la órden.
