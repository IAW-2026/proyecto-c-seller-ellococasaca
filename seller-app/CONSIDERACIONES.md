# El Loco Casaca - Seller App - Consideraciones 👕

A continuación se detallarán ciertas consideraciones a tener en cuenta sobre el proyecto, así como también contemplar ciertas **limitaciones** y **funcionalidades pendientes** en el mismo.

## 🛠️ Limitaciones
- En esta entrega, si bien en la interfaz de subir y editar productos se encuentra el campo para subir imágenes, el mismo no fue desarrollado y quedará para desarrollar en la última etapa del proyecto. Cabe aclarar que se hará uso de **Cloudinary** para gestionar las subidas y bajadas de imágenes.

## 🛠️ Funcionalidades y Features Pendientes
- Implementación de subida y bajada de imágenes.
- Unificar estilos visuales entre páginas, más precisamente, en la de **Visualizar Productos**.
- Mejorar el manejo de errores.
- Refactorizar componentes reutilizables, como lo pueden ser **create-product-form** y **update-product-form**.
- Implementar loading para mejorar la experiencia del usuario.
